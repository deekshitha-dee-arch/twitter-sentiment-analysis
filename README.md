# Twitter Sentiment Analysis

## Overview

This project demonstrates sentiment analysis on Twitter data using Natural Language Processing (NLP) techniques in R.

The goal was to analyze public opinion and identify positive, negative, and neutral sentiment patterns from social media text data.

## Technologies Used

- R
- NLP
- tidytext
- syuzhet
- ggplot2

## Project Workflow

1. Data Collection
2. Data Cleaning
3. Text Tokenization
4. Sentiment Scoring
5. Visualization
6. Insight Generation

## Sentiment Methods

### AFINN Lexicon
Used for numerical sentiment scoring.

### Bing Lexicon
Used for positive and negative classification.

## Key Skills Demonstrated

- Natural Language Processing
- Text Mining
- Sentiment Analysis
- Data Cleaning
- Data Visualization
- R Programming

## Project Outcome

Successfully developed a sentiment analysis pipeline capable of extracting insights from Twitter data and visualizing public opinion trends.
